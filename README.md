# EdgeFinder - R&D Exploration Platform

A comprehensive platform for systematic research exploration and innovation discovery, built on "The Genesis of Technoscientific Revolutions" framework.

## 🚀 Live Demo

**Deployed Application:** https://edgefinder-r-d-explo-v4nu.bolt.host

## 🎯 Overview

EdgeFinder is an intelligent research exploration platform that helps researchers, innovators, and organizations systematically discover breakthrough opportunities at the edges of knowledge. It combines human-centered design methods with AI-powered insights to navigate the complex landscape of innovation.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/edgefinder.git
cd edgefinder
```

2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**
```bash
cp .env.example .env
```

Edit `.env` with your API keys:
```env
# EdgeFinder Configuration
VITE_EDGEFINDER_API_KEY=your_edgefinder_api_key_here

# SuperConnector Integration
VITE_SUPERCONNECTOR_API_URL=https://api.superconnector.network/v1
VITE_SUPERCONNECTOR_CLIENT_ID=your_superconnector_client_id

# Convergent Research Integration  
VITE_CONVERGENT_RESEARCH_API_KEY=your_convergent_research_api_key
```

4. **Start development server**
```bash
npm run dev
```

5. **Open in browser**
Navigate to `http://localhost:5173`
